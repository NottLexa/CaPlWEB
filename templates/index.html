What is Casual Playground?

Casual Playground (CaPl in short) is a customizable grid-based 2D-sandbox inspired by cellular automatons. Casual Playground contains all needed instruments for a cellular automata user to run any kind of a cellular automaton.

Natively Casual Playground contains John Conway's Game of Life and WireWorld cells to put on a board, but Casual Playground (desktop client only (now)) also has a modding support, so that any player can install more custom cellular automata or write his own ones.

With the help of Casual Playgrounds' own programming language – Casual Playground Language (CPL) – writing your own mods for CaPl becomes very simple. Check CPL Scripting Introduction on GitHub to begin writing mods.

I've been making this game all by myself for a few years in my free time. All of its code was written in native JavaScript, using Node.JS environment for a desktop client that may work offline. Casual Playground's source code is available on GitHub, you can contribute to the project, or download release from there.
